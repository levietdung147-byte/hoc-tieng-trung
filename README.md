# 学中文 · Học Tiếng Trung — 30 phút mỗi ngày

App học tiếng Trung HSK 1–2, mỗi ngày 30 phút. HTML/CSS/JS thuần, không build tool.

**Học trên điện thoại:** mở link GitHub Pages → bấm chia sẻ → *Thêm vào màn hình chính* để dùng như một app.

## Tính năng
- 📇 **Học từ mới** — flashcard lật (chữ Hán → pinyin + nghĩa), nghe phát âm (Web Speech API).
- ✍️ **Luyện viết** — viết chữ Hán theo nét bằng [Hanzi Writer](https://hanziwriter.org/).
- 🎯 **Ôn tập** — quiz trắc nghiệm nghĩa.
- 🔥 Streak, heatmap 30 ngày, hẹn giờ 30 phút. Tiến độ lưu trên máy (localStorage).
